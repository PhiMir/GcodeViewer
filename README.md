# Gcode 3D Viewer

3D 프린터 Gcode를 시각화하고 실행 시뮬레이션을 할 수 있는 웹 기반 툴입니다.

## 주요 기능

- ✅ Gcode 파일 업로드 및 파싱
- ✅ 3D 경로 시각화 (Three.js)
- ✅ 레이어별 색상 구분
- ✅ 애니메이션 재생/일시정지
- ✅ 재생 속도 조절 (1x ~ 10x)
- ✅ 진행도 슬라이더
- ✅ 이동 경로 표시 옵션
- ✅ 실시간 통계 (레이어 수, 이동 횟수, 거리 등)
- ✅ 인터랙티브 3D 컨트롤 (회전, 줌, 팬)

## 설치 방법

```bash
# 패키지 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 사용 방법

1. 브라우저에서 개발 서버 열기 (기본: http://localhost:5173)
2. "파일 선택" 버튼으로 Gcode 파일 업로드 (.gcode, .nc, .txt)
3. 3D 뷰에서 경로 확인
4. 재생 버튼으로 애니메이션 시작
5. 마우스로 3D 모델 조작:
   - 좌클릭 드래그: 회전
   - 마우스 휠: 줌
   - 우클릭 드래그: 이동

## 지원 Gcode 명령어

- G0, G1: 직선 이동
- G90: 절대 좌표 모드
- G91: 상대 좌표 모드
- G28: 홈 포지션
- G92: 현재 위치 설정

## 기술 스택

- React 18
- Three.js (3D 렌더링)
- Vite (빌드 도구)
- Tailwind CSS (스타일링)
